# Brick Breaker - UI/UX 설계

## 전체 레이아웃

```
┌─────────────────────────────────────────────────────────┐
│                      HEADER (60px)                      │
│  Brick Breaker              점수: 0000   생명: ♥♥♥    │
├─────────────────────────────────────────────────────────┤
│                                                         │
│                                                         │
│                                                         │
│                   GAME CANVAS                           │
│                    (800x600)                            │
│                                                         │
│                                                         │
│                                                         │
├─────────────────────────────────────────────────────────┤
│                      FOOTER (40px)                      │
│              Stage: 1    |    High Score: 0000          │
└─────────────────────────────────────────────────────────┘
```

**전체 크기:** 800 x 700 px (캔버스 600 + 헤더 60 + 푸터 40)

## 화면별 상세

### 1. 시작 화면 (idle)

```
┌─────────────────────────────────────────┐
│                                         │
│                                         │
│           BRICK BREAKER                 │  ← 타이틀 (3xl, bold)
│                                         │
│           Click or Press SPACE          │  ← 안내 (lg, gray-400)
│               to Start                  │
│                                         │
│                                         │
│           High Score: 1234              │  ← 하이스코어 표시
│                                         │
└─────────────────────────────────────────┘
```

**동작:**
- 클릭 또는 Space → playing 상태로 전환

### 2. 플레이 화면 (playing)

```
┌─────────────────────────────────────────┐
│ Score: 0150        Stage: 1    ♥♥♥     │  ← HUD
├─────────────────────────────────────────┤
│ ■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■  │  ← 벽돌
│ ■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■  │
│ ■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■  │
│                                         │
│                    ●                    │  ← 공
│                                         │
│                                         │
│                                         │
│               ════════                  │  ← 패들
└─────────────────────────────────────────┘
```

**HUD 요소:**
- 왼쪽: 현재 점수
- 중앙: 스테이지 번호
- 오른쪽: 생명 (하트 아이콘)

### 3. 일시정지 화면 (paused)

```
┌─────────────────────────────────────────┐
│                                         │
│     ┌───────────────────────────┐       │
│     │                           │       │
│     │         PAUSED            │       │  ← 오버레이
│     │                           │       │
│     │   Press ESC to Resume     │       │
│     │                           │       │
│     └───────────────────────────┘       │
│                                         │
└─────────────────────────────────────────┘
```

**동작:**
- 게임 화면 위에 반투명 오버레이
- ESC 또는 P → playing 상태로 복귀

### 4. 게임오버 화면 (gameover)

```
┌─────────────────────────────────────────┐
│                                         │
│              GAME OVER                  │  ← 타이틀 (red-500)
│                                         │
│           Final Score: 1234             │
│                                         │
│     ┌─────────────────────────┐         │
│     │  Enter your name:       │         │  ← 이름 입력
│     │  [_______________]      │         │
│     │                         │         │
│     │   [Submit]  [Retry]     │         │  ← 버튼
│     └─────────────────────────┘         │
│                                         │
└─────────────────────────────────────────┘
```

**동작:**
- 하이스코어 등록 가능 시 이름 입력
- Submit: 점수 저장 후 idle로
- Retry: 바로 재시작

### 5. 스테이지 클리어 화면 (clear)

```
┌─────────────────────────────────────────┐
│                                         │
│            STAGE CLEAR!                 │  ← 타이틀 (green-500)
│                                         │
│           Score: 1234                   │
│           Bonus: +500                   │
│                                         │
│        Press SPACE for Next Stage       │
│                                         │
└─────────────────────────────────────────┘
```

**동작:**
- 2초 후 자동 또는 Space로 다음 스테이지

## 컴포넌트 상세

### 버튼

```
┌─────────────────┐
│     Button      │   기본 상태
└─────────────────┘
 bg-blue-500, text-white, rounded-lg, px-6, py-2

┌─────────────────┐
│     Button      │   호버 상태
└─────────────────┘
 bg-blue-600

┌─────────────────┐
│     Button      │   비활성화
└─────────────────┘
 bg-gray-500, cursor-not-allowed
```

### 입력 필드

```
┌─────────────────────────┐
│ Placeholder...          │
└─────────────────────────┘
 bg-gray-800, border-gray-600, text-white, rounded-lg
 focus: border-blue-500
```

### 생명 표시

```
♥♥♥   → 3개 (red-500)
♥♥♡   → 2개 (red-500 + gray-600)
♥♡♡   → 1개
```

## 반응형 고려

| 화면 | 처리 |
|------|------|
| 800px 미만 | 캔버스 비율 유지, 축소 표시 |
| 800px 이상 | 원본 크기 중앙 정렬 |

```css
/* 캔버스 컨테이너 */
.game-container {
  max-width: 800px;
  margin: 0 auto;
}
```

## 애니메이션

| 요소 | 애니메이션 | 지속시간 |
|------|------------|----------|
| 시작 화면 텍스트 | 깜빡임 (pulse) | 2s |
| 버튼 호버 | 색상 전환 | 150ms |
| 오버레이 등장 | 페이드인 | 200ms |
| 점수 증가 | 숫자 카운트업 | 300ms |

## 접근성

- 키보드만으로 모든 조작 가능
- 포커스 표시 명확히
- 색상 대비 4.5:1 이상 유지
